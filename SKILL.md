# Скилл: Аудит расходов

## Описание

Анализирует CSV-файл с расходами компании или магазина. Находит общую сумму, разбивку по категориям, дубликаты, аномальные траты и отрицательные суммы. Пишет краткую сводку простым языком для владельца.

## Как использовать

Пользователь передаёт путь к файлу или URL:

```
python audit.py expenses.csv
python audit.py https://example.com/data/expenses.csv
```

Результат сохраняется в `report.txt` рядом со скриптом.

---

## Что делает скилл (по шагам)

### 1. Получить файл
- Если передан **URL** — скачать файл автоматически
- Если передан **путь** — читать напрямую

### 2. Определить структуру файла
Скрипт автоматически определяет нужные столбцы по названию (на русском или английском):

| Столбец | Ищет названия |
|---|---|
| Дата | `date`, `дата`, `data` |
| Категория | `category`, `категория`, `cat`, `тип`, `вид` |
| Сумма | `amount`, `сумма`, `sum`, `total`, `итого`, `price`, `цена` |
| Описание | `description`, `описание`, `desc`, `наименование`, `название`, `name` |

### 3. Запустить анализ
Скрипт считает:
- Общую сумму расходов
- Сумму по каждой категории (от большей к меньшей)
- Топ-5 самых крупных трат
- Дубликаты: строки с одинаковой датой, описанием и суммой
- Аномалии: суммы, которые резко выбиваются из нормы для своей категории (более 3 стандартных отклонений от среднего)
- Строки с отрицательными суммами

### 4. Написать сводку
После анализа — написать краткую сводку для владельца **простым языком, без технических терминов**:

- Что не так с расходами
- На что обратить внимание
- Где возможны ошибки или мошенничество
- Какие категории съедают больше всего денег

### 5. Показать результат
- Вывести сводку пользователю
- Указать, что полный отчёт сохранён в `report.txt`

## Требования

- Python 3.8+
- Стандартная библиотека (csv, math, urllib — ничего устанавливать не нужно)
- Файл должен быть в формате CSV с заголовками

## Параметры

| Параметр | Описание | Пример |
|---|---|---|
| `<файл или URL>` | Путь к CSV-файлу или ссылка на него | `expenses.csv` или `https://...` |
